## Getting started
```bash
#npm
npm install
npm start # default port 3000

#yarn 
yarn 
yarn start

#make (first of all install dependencies)
make start

#docker - port 3001
docker build -t testapp.image .
docker run -d -p 3001:3001 --rm --name testapp.cont testapp.image
```
